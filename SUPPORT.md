# Hỗ trợ

Cần trợ giúp với enrol_sepay? Các kênh:

- **Câu hỏi / cách dùng**: mở issue với nhãn `question`, hoặc [Discussion](https://github.com/MinhThang1009/sepay-plugin-moodle/discussions) nếu repo đã bật Discussions.
- **Báo lỗi**: mở issue theo mẫu Bug report.
- **Đề xuất tính năng**: mở issue theo mẫu Feature request.
- **Bảo mật**: KHÔNG mở issue công khai — xem [SECURITY.md](SECURITY.md).
